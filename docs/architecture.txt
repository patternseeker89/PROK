1. Написать консольную версию приложения.
2. Визуализация это будет отдельная программа, возможно на другом языке.
3. Возможно в будущем можно будет применять скрипты для отдельных нод в дереве.
Например есть нода депозита, и в нем есть поле даты окончания, написав скрипт, который будет отрабатывать во время окончания депозита.
И делать какие-то действия. Например оповещать меня об этом отправкой сообщения на почту.
4. Скорее всего удалать информацию будет нельзя, только добавлять или изменять, или скрывать.
Полного удаления не будет.

Типы нод:
1. Обычная
2. Секретная
3. Архивная
4. Скрытная
